# TAX360 assets

Branded infographics + logo for the TAX360 daily tax-education posts (SHD Advisory).

## House style (brand-aligned, modern) — see `templates/tax360_card_example.py`
- **Palette (from the SHD logo teal #3C8CA0):** deep-petrol background `#0B2731`,
  panels `#11343F`/`#0E2C36`, brand teal `#3C8CA0`, bright teal `#5BB6CC`,
  gold `#E7B24C` (numbers / single accent), white text, muted `#9DBBC4`,
  hairlines `#1E4A58`. Do NOT use the old navy/gold palette.
- **Type:** Poppins (Bold headlines/numbers, Medium subheads), Lato
  (Bold letter-spaced kickers/labels, Regular body, Light hero description).
- **Layout:** header (TAX360 wordmark + "Educational Tax Tip" + white logo badge
  top-right) · letter-spaced kicker · big headline · gold subhead · muted
  description · stat band split by hairlines · 2 comparison cards (accent
  underline + dot bullets) · slim accent strip · caveat · footer.
- **Rules:** no post date on the card (evergreen); 64px safe margin; measure +
  auto-fit text so nothing bleeds; 1080x1350; keep PNG < ~1MB.

Daily cards are committed as `images/tax360-YYYY-MM-DD.png`.
